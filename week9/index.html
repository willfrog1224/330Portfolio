~9~

Browser Object Model (or BOM for short) is a collection of properties and methods that contain information about the browser and computer screen

Global - const, let or var keywords

Global variables can be accessed in all parts of the program

three functions that produced dialogs in the browsers: alert(), confirm() and prompt()

window.alert() method will pause the execution of the program and display a message in a dialog box

window object has a number of properties and methods that provide information about the users browser

window.location property is an object that contains information about the URL of the current page

protocol property returns a string describing the protocol used (such as http, https, pop2, ftp etc.). Note that there is a colon (:) at the end

host property returns a string describing the domain of the current URL and the port number

hostname property returns a string describing the domain of the current URL

port property returns a string describing the port number

pathname property returns a string of the path that follows the domain

window.location object also has the following methods

reload() method can be used to force a reload of the current page

assign() method can be used to load another resource from a URL provided as a parameter

window.screen object contains information about the screen the browser is displayed on

colorDepth property can be used to find the color bit depth of the users monitor

Cookies are session cookies by default. This means they are deleted once a browser session is finished

window object provides some useful methods for scheduling the execution of a function, and for repeatedly executing functions at regular intervals

window.setInterval() method works in a similar way to window.setTimeout(), except that it will repeatedly invoke the callback function after every given number of milliseconds

setTimeOut() and setInterval() methods can be used to animate elements

method of the window object works in much the same way as the window.setInterval() method

window.requestAnimationFrame() method can be used to produce smooth and optimized animation by utilizing the browser's built-in graphics capabilities. It can be canceled using the cancelAnimationFrame() method

~14~

data- attribute is a way of embedding data in a web page using custom attributes that are ignored by the browser

Start with data-

Contain only lowercase letters, numbers, hyphens, dots, colons or underscores

Include an optional string value

data-dropdown could be identified as dropdown menu.

TML5 specification contains a number of APIs that help to gain access to hardware, such as cameras, batteries, geolocation, and the graphics card

Web Storage API has some crucial differences with cookies

Information stored is not shared with the server on every request

Information is available in multiple windows of the browser (but only if the domain is the same)

Storage capacity limit is much larger than the 4KB limit for cookies

Any data stored does not automatically expire as it does with cookies

key - key of the item that changed

newValue - new value to which it has been changed

oldValue - previous value before it was changed

storageArea - if it is stored in local or session storage

Geolocation API is used to obtain the geographical position of the device

position object passed to the youAreHere() function has a coords property with a latitude and longitude property

position object has several other properties that can be used to find out information about the location and movement of the device

clearWatch() method can be used to stop the callback being called

Notification API allows you to show messages using the system's notifications

Audio and video clips also have a number of events that will fire when they occur

canvas element was introduced to allow graphics to be drawn onto a web page in real time using JavaScript

A shim or polyfill is a piece of code that adds support of missing features to older browsers
