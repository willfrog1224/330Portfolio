CSS3 transform property lets you lets you translate, rotate, scale, and/or skew any element on the page

translate(x,y) function moves an element x from the left, and y from the top

along the x axis - transform: translateX(45px);

along the y axis - transform: translateY(-30px);

Transforms dont work on inline elements

scale(x,y) function scales an element by the defined factors horizontally then vertically

translate, you can also use the scaleX(x) or scaleY(y) functions

rotate() function rotates an element around the point of origin by a specified angle value

skew(x,y) function specifies a skew along the x and y axes

steps to create a simple transition using only CSS

Declare the original state of the element in the default style declaration

Declare the final state of your transitioned element; for example, a :hover state

Include the transition functions in your default style declaration using the transition properties

transition-property property defines the CSS properties of the element that should be transitioned, with all for all properties being the default

exception to this “if there is a valid midpoint, it can be transitioned” rule is visibility

transition-duration property sets how long the transition will take: the duration of time it takes to go from the default state to the transitioned state

transition-timing-function lets you control the pace of the transition in even more granular detail

transition-delay - introduce a delay before the transition begins

transition properties allow for multiple transitions in one call

Animations - allow you to control each step of an animation via keyframes

To animate an element in CSS, you first create a named animation, then attach it to an element in that elements property declaration block

animation-iteration-count property lets you define how many times the animation will play through

animation iterates, it normally goes from the 0% to the 100% keyframe
